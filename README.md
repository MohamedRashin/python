This project is an ML-based Smart Farming System that uses live location data and soil parameters to provide accurate crop recommendations and intelligent irrigation control.

By applying Machine Learning algorithms, the system predicts the most suitable crop and determines optimal irrigation needs, helping farmers increase yield and reduce resource wastage.

🎯 Objectives
Collect real-time soil and environmental data using location
Use Machine Learning to predict suitable crops
Optimize irrigation based on soil moisture prediction
Improve productivity and reduce water usage
⚙️ Features
📍 Location-based soil data analysis
🤖 ML Crop Recommendation System
💧 Smart irrigation prediction using ML
📊 Data visualization dashboard
⚡ Real-time decision support
🧠 Machine Learning Model
Algorithm Used:
Random Forest / Decision Tree / XGBoost (you can choose one)
Input Features:
Soil pH
Temperature
Humidity
Rainfall
Nitrogen (N), Phosphorus (P), Potassium (K)
Output:
Recommended Crop
Irrigation Requirement
🛠️ Technologies Used
Frontend: Flutter / Web
Backend: Python (Flask / FastAPI)
ML Libraries:
Scikit-learn
Pandas
NumPy
Database: Firebase / MySQL
🔄 Workflow
User location is captured
Soil and weather data are collected
Data is sent to ML model
Model predicts:
Best crop to grow 🌾
Irrigation requirement 💧
Results displayed to user
📊 Dataset
Soil dataset with NPK values
Weather dataset (temperature, rainfall, humidity)
Crop yield dataset
